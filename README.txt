#Revisando ArrayList dentre outras coisas

Estou revisando praticamente tudo de Java pois quero criar o TCC com mais precisão e da melhor maneira possível
Em breve adentrarei dentro de Spring Boot, assim podendo finalmente dar inicio ao projeto

Esse projeto foi bem importante, nem pelas listas em si, e sim pela relação entre as classes, quem ou o que faz cada coisa.
Nesse caso, o Cartão é quem realiza a Compra, já a Main é somente a interface do usuário, a Main não sabe o que o Cartão ou o que a Conta fazem,
somente sabe que eles existem. Isso serve também para o Cartão e para a Compra, assim gerando mais segurança para o projeto.

Estou sempre aberto a sugestões do que melhorar no código, aceito dicas e feedbacks, dos mais simples aos mais avançados.
